---
title: Headless API
description: Imperative API for building scenes outside React / Vue.
---

The `glyphcss` vanilla package exposes a polycss-shaped imperative API plus the raw
`glyphcss-core` math primitives.

## Scene factory

```ts
import { createGlyphcssScene } from "glyphcss";

const host = document.querySelector("#scene")!;
const scene = createGlyphcssScene(host, {
  mesh: "/cottage.glb",        // path | ArrayBuffer | triangle list
  rotX: 0.4,
  rotY: 0,
  scale: 0.32,
  stretch: 0.95,
  distance: 3,
  mode: "wireframe",            // | "solid" | "voxel"
  animate: { axis: "y", durationMs: 6000 },
});

scene.addHotspot(
  { id: "roof", at: [0, 1.2, 0], size: [3, 2] },
  () => alert("roof clicked"),
);

scene.dispose();
```

## Camera factories

```ts
import { createGlyphcssPerspectiveCamera, createGlyphcssOrthographicCamera } from "glyphcss";

const perspective = createGlyphcssPerspectiveCamera({
  rotX: 0.4, rotY: 0, distance: 3, scale: 0.32, stretch: 0.95,
});

const ortho = createGlyphcssOrthographicCamera({ rotX: 0.4, zoom: 0.4 });

// All five state fields are mutable after construction:
perspective.rotY = Math.PI / 4;
perspective.distance = 4;
```

## Lower-level pieces (`glyphcss-core`)

```ts
import {
  buildSceneContext,
  rasterize,
  bakeFrames,
  projectHotspots,
  parseObj, parseGltf, parseVox, loadMesh,
} from "glyphcss-core";

const triangles = await loadMesh("/cottage.glb");
const camera = createGlyphcssPerspectiveCamera({ /* ... */ });
const grid = { cols: 160, rows: 48, cellAspect: 2.0 };

const scene = buildSceneContext({ camera, grid, triangles, mode: "solid" });

const frames = bakeFrames(scene, 60, "y");
// frames is string[] — stack them into a <pre> and animate via CSS steps(60).

const cells = projectHotspots(hotspots, camera, 160, 48, 2.0);
// cells is { id, col, row, depth, visible }[]
```

`glyphcss-core` is renderer-agnostic and has zero browser globals — you can run it in
Node or a web worker. The DOM construction lives in `glyphcss` (vanilla) and the
framework packages.
