---
title: Core Types
description: TypeScript type definitions for polygons, camera state, lighting, and parse results.
---

All types are exported from `@polycss/react`, `@polycss/vue`, and `@polycss/core`.

## `Polygon`

Represents a single polygon (triangle or quad) in 3D space.

```ts
interface Polygon {
  vertices: [number, number, number][];  // Required — 3+ points in world space
  color?: string;                        // CSS color value
  texture?: string;                      // Image URL for UV-mapped face
  uvs?: [number, number][];             // UV coordinates (one per vertex)
  normal?: [number, number, number];    // Surface normal for lighting
  data?: Record<string, unknown>;       // User-defined metadata
}
```

---

## `DirectionalLight`

Controls directional and ambient lighting for the scene.

```ts
interface DirectionalLight {
  direction: [number, number, number];  // Direction FROM surface TO light (normalized)
  color?: string;                       // Directional light color (default: "#ffffff")
  ambient?: number;                     // Ambient intensity 0–1 (default: 0.4)
  ambientColor?: string;               // Ambient light color (default: "#ffffff")
}
```

## `ParseResult`

Unified return shape from all mesh parsers (`parseObj`, `parseGltf`).

```ts
interface ParseResult {
  polygons: Polygon[];          // Parsed polygon array
  objectUrls: string[];         // Blob URLs minted for embedded textures
  dispose: () => void;          // Revoke all objectUrls (call on unmount)
  warnings: string[];           // Non-fatal parse warnings
}
```

---

## `ObjParseOptions`

Options for `parseObj`.

```ts
interface ObjParseOptions {
  targetSize?: number;                          // Scale model to this world-unit size
  defaultColor?: string;                        // Fallback color for un-colored faces
  materialColors?: Record<string, string>;      // Material name → CSS color overrides
  materialTextures?: Record<string, string>;    // Material name → image URL overrides
  includeObjects?: string[];                    // Only include these named objects
  excludeObjects?: string[];                    // Exclude these named objects
  palette?: string[];                           // Quantize colors to this palette
}
```

---

## `GltfParseOptions`

Options for `parseGltf`.

```ts
interface GltfParseOptions {
  targetSize?: number;   // Scale model to this world-unit size
  baseUrl?: string;      // Base URL for resolving external texture references
}
```

---

## `AutoRotateOption`

Configures automatic camera rotation.

```ts
type AutoRotateOption =
  | boolean
  | number
  | {
      axis?: "x" | "y";
      speed?: number;
      pauseOnInteraction?: boolean;
    };
```

---

## `ProjectionMode`

Projection style (currently isometric-like via CSS perspective).

```ts
type ProjectionMode = "perspective" | "orthographic";
```
