title: PolyScene
description: Scene component that sets up the 3D viewport, camera, and lighting for polygon-mesh rendering.
---

import { Tabs, TabItem } from '@astrojs/starlight/components';

`PolyScene` is the root of every polycss render tree. It sets up CSS 3D perspective, accepts camera props, applies directional lighting, and renders its children (typically `PolyMesh` or `Poly` elements) in 3D space.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `perspective` | `number` | `1000` | CSS perspective distance in pixels. Higher values feel flatter (more isometric); lower values exaggerate depth. |
| `rotX` | `number` | `65` | Camera rotation around the X axis in degrees. |
| `rotY` | `number` | `45` | Camera rotation around the Y axis in degrees (0–360). |
| `directionalLight` | `DirectionalLight` | — | Directional light source. See [DirectionalLight](#directionallight). |
| `polygons` | `Polygon[]` | — | Optional flat array of polygon objects rendered as direct children. Composes with JSX children. |
| `merge` | `"off" \| "auto"` | `"off"` | Merge strategy. `"auto"` merges coplanar same-material polygons before rendering. |
| `children` | `ReactNode` | — | `PolyMesh` and/or `Poly` components. |

## DirectionalLight

```ts
interface DirectionalLight {
  direction: [number, number, number];  // Normalized direction vector
  color?: string;                       // Light color (default: "#ffffff")
  ambient?: number;                     // Ambient intensity 0–1 (default: 0.4)
}
```

## Usage

### Basic Scene

A scene with a single GLB mesh at default camera angle.

### Scene with Camera and Lighting

```tsx
<PolyScene
  perspective={1000}
  rotX={65}
  rotY={45}
  directionalLight={{
    direction: [0.5, -0.7, 0.6],
    color: "#ffe4a8",
    ambient: 0.4,
  }}
>
  <PolyMesh src="/cottage.glb" position={[0, 0, 0]} />
  <PolyMesh src="/tree.glb"    position={[8, 0, 0]} scale={0.5} />
</PolyScene>
```

### Multiple Meshes

```tsx
<PolyScene rotX={65} rotY={45}>
  <PolyMesh src="/cottage.glb" position={[0, 0, 0]} />
  <PolyMesh src="/tree.glb"    position={[8, 0, 0]} scale={0.5} />
</PolyScene>
```

### Flat Polygon Array

Pass a `polygons` array directly to render static geometry without a file loader.

```tsx
const polygons: Polygon[] = [
  { vertices: [[0,0,0],[1,0,0],[0,1,0]], color: "#f00" },
  { vertices: [[2,0,0],[3,0,0],[2,1,0]], color: "#00f" },
];

<PolyScene polygons={polygons} />
```
