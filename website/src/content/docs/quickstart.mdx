## Hello cuboctahedron

<Tabs syncKey="fw">
  <TabItem label="React">

```tsx
import { GlyphcssCamera, GlyphcssScene, GlyphcssOrbitControls, GlyphcssMesh, GlyphcssHotspot } from "@glyphcss/react";

export function App() {
  return (
    <GlyphcssCamera rotX={0.4} scale={0.32}>
      <GlyphcssScene>
        <GlyphcssOrbitControls drag wheel />
        <GlyphcssMesh geometry="cuboctahedron">
          <GlyphcssHotspot at={[1, 0, 0]} onClick={() => alert("vertex")} />
        </GlyphcssMesh>
      </GlyphcssScene>
    </GlyphcssCamera>
  );
}
```

  </TabItem>

## What you get

- A single `<pre>` rendered as ASCII glyphs, animating a full rotation in ~6 seconds.
- One absolutely-positioned `<div role="button">` for each `<GlyphcssHotspot>`,
  tracking the rotating mesh exactly (per-frame keyframes synced via `steps(N)`).
- Drag-to-rotate and scroll-to-zoom out of the box via `<GlyphcssOrbitControls>`.
