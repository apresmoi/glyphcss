VoxCSS is a **CSS voxel engine** — a 3D grid for the DOM. It renders HTML cuboids by stacking CSS grid layers and applying CSS transforms. No `<canvas>`, no WebGL, no dependencies — just pure DOM and CSS.

Every voxel is a real HTML element. You can inspect it in DevTools, style it with CSS, and interact with it using standard DOM events. VoxCSS brings the simplicity of pixel art into three dimensions, right inside your web page.

## Framework Support

VoxCSS ships with first-class bindings for **React** and **Vue**, plus a vanilla JavaScript API that works anywhere. Pick whatever fits your stack.

You can also load VoxCSS directly from a CDN with no build step:

```html
<script type="module">
  import { renderScene } from "https://unpkg.com/@layoutit/voxcss";
</script>
```

## A quick taste

A single orange voxel with an interactive camera. Drag it to rotate.

```tsx
import { VoxCamera, VoxScene } from "@layoutit/voxcss/react";

const voxels = [{ x: 1, y: 1, z: 0, color: "#f97316" }];

export function App() {
  return (
    <VoxCamera interactive>
      <VoxScene voxels={voxels} />
    </VoxCamera>
  );
}
```
